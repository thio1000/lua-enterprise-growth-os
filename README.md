# Lua Enterprise Growth OS — Kenya

A self-contained, interactive strategy system built as a work sample for Lua Global's Commercial Lead, Kenya role: 20 sections covering market intelligence, buying committees, digital employees, commercial/growth motion, North Star metrics, and a 90-day execution plan for opening the Kenya market.

Every claim is labeled sourced / reasoned / no-data-found — see the Appendix section inside the app for the full source list.

## Deploy

This is a single static `index.html` with everything (fonts, styles, script) inlined — no build step, no dependencies.

**Netlify:**
1. Push this repo to GitHub
2. In Netlify: "Add new site" → "Import an existing project" → pick this repo
3. Build command: leave blank
4. Publish directory: `.`
5. Deploy

**Or drag-and-drop:** Netlify also supports dragging this folder straight onto app.netlify.com/drop for an instant deploy without a repo.
